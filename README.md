# 🐕 Paw Planner 
Esta web registra y organiza los turnos en los que un grupo de tres personas deben sacar a pasear a su perro.

## Uso
Actualmente, se puede visitar la página de uso personal a traves de [este enlace](https://pawplanner.streamlit.app), aunque sin la contraseña no permite hacer modificaciones.
Con los scripts del repositorio y unas hojas de cálculo de google con este [formato](https://docs.google.com/spreadsheets/d/1yZX97GG074dfIxb5BHap3ExqcPTX1LyPHHONPkgHkEk/edit?gid=1968271575#gid=1968271575) se puede utilizar sin problema subiendolo a la nube o ejecutando el script visual_st.py.
